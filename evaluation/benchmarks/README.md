## Benchmarks Assets

This directory copies benchmark assets from the repository's original `eval/`
directory so the `evaluation/` workflow is self-contained.

- `alpacaeval2/`: model configs and prompt templates
- `arenahard/`: model configs and prompt templates
- `mt-bench/`: GPT-4 Turbo reference answers file

Use `python evaluation/run_benchmarks.py --help` for benchmark command
wrappers.
